# Krishna Prasath - Portfolio Website

A modern, responsive portfolio website built with Next.js 15, React 19, and Tailwind CSS 4.

## Features

- **Modern Design**: Clean and elegant UI with smooth animations
- **Responsive**: Fully responsive design that works on all devices
- **Fast**: Built with Next.js for optimal performance
- **Smooth Scrolling**: Integrated Lenis for buttery smooth scrolling experience
- **Animations**: Motion library for engaging animations and transitions

## Tech Stack

- **Framework**: Next.js 15.2.2
- **React**: 19.0.0
- **Styling**: Tailwind CSS 4
- **Animations**: Motion (Framer Motion)
- **Smooth Scrolling**: Lenis
- **Language**: TypeScript

## Getting Started

### Prerequisites

- Node.js 20 or higher
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd krishnaprasath-portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

```
├── app/                    # Next.js app directory
│   ├── layout.tsx         # Root layout
│   ├── page.tsx           # Home page
│   └── globals.css        # Global styles
├── components/            # React components
│   ├── Client/           # Client-side components
│   ├── Server/           # Server-side components
│   └── SVGComponents/    # SVG icon components
├── sections/             # Page sections
│   ├── Hero/            # Hero section
│   ├── Introduction/    # Introduction section
│   ├── WellnessSanctuary/ # Philosophy section
│   └── Footer/          # Footer section
└── public/              # Static assets

```

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm start` - Start production server
- `npm run lint` - Run ESLint

## Customization

You can customize the portfolio by:

1. **Content**: Edit the content in `sections/` directory
2. **Styling**: Modify Tailwind classes or global styles
3. **Images**: Replace images in `public/` directory
4. **Colors**: Update color schemes in component files
